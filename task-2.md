# Задача 2. Публикация своего товара на маркетплейсе

Я, как авторизированный и верифицированный продавец, хочу опубликовать свои товары на маркетплейсе, чтобы мои товары стали доступны для покупки пользователям, и я мог их продавать.

## Критерии приемки

- Опубликовать карточку товара можно только после ввода обязательных атрибутов товара.
- Карточка товара проходит модерацию после публикации.
- Блокировать создание карточек, если аккаунт под санкциями.
- Карточка имеет статусы: `опубликован`, `на модерации`, `отклонен`, `остановлен`.
- Запуск товара в продажу невозможен, пока остаток товара нулевой.

## Основной сценарий (Use Case)

- **Актор:** Пользователь
- **Результат:** Созданная карточка товара
- **Предусловия:** Пользователь авторизован как продавец

### Основной поток

1. Пользователь открывает страницу добавления нового товара.
2. Пользователь заполняет данные: название, описание, категория, цена, характеристики.
3. Система валидирует заполненные поля.
4. Пользователь загружает медиафайлы в специальную форму с ячейками под каждое медиа (фото/видео).
5. Система подгружает медиа и отображает превью карточки.
6. Пользователь сохраняет карточку.
7. Система сохраняет карточку со статусом `На модерации`.
8. Пользователя перенаправляет на страницу всех созданных карточек.

### Альтернативные сценарии

- Не заполнены обязательные поля / неверно заполнены поля: система подсвечивает проблемные поля и показывает сообщение об ошибке.
- Ошибка загрузки изображения: система показывает модальное окно с ошибкой.
- Карточка не проходит модерацию: продавец получает уведомление с комментариями модератора.
- Возможность массового создания карточек путем загрузки Excel-таблицы с данными.
